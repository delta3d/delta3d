Uses modules: dtCore, dtABC, dtUtil, dtGame, dtDIS

Purpose
-------
To demonstrate how to configure and use dtDIS for incoming DIS PDU's.
See the Connection.xml and actormap.xml to define the socket connection
and the mapping between DIS EntityType and Delta3D ActorTypes.

Note, not all incoming DIS PDU types are supported and only a few of the 
DIS attributes are being used in this example.


Controls
--------

Esc    -  Exit
alt-Left Mouse Button   -  Rotate View
Middle Mouse Button  -  Zoom View
