########################################

Uses modules: dtCore, dtABC, dtUtil

Purpose
-------
To demonstrate basic functionality of the dtABC::ProximityTrigger
& dtABC::Action classes. Just walk into the light!

Instructions
------------
Win32 Source: 
- Open VisualStudio\examples\testTrigger\testTrigger.sln.
- Build testTrigger.
- Start testTrigger.

Win32 Binary:
- Go to bin.
- Double-click testTrigger.exe.

Linux Source:
- To build only testTrigger, from the root delta3d folder:
  scons bin/testTrigger

Linux Binary:
- cd bin
- ./testTrigger

Controls
---------
Esc - exit
