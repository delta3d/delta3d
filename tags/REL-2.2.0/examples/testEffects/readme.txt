Uses modules: dtCore, dtABC

Purpose
-------
To demonstrate particle effects. This example shows a smoke particle effect as
well as an explosion particle effect.

Instructions
------------
Win32 Source: 
- Open VisualStudio\examples\testEffects\testEffects.sln.
- Build testEffects.
- Start testEffects.

Win32 Binary:
- Go to bin.
- Double-click testEffects.exe.

Linux Source:
- To build only testEffects, from the root delta3d folder:
  scons bin/testEffects

Linux Binary:
- cd bin
- ./testEffects

Controls
--------
Up     -  Rotate camera up
Down   -  Rotate camera down
Left   -  Rotate camera left
Right  -  Rotate camera right
Esc    -  Exit
