# Delta3D Examples: testPhysics        #
#                                      #
########################################

Uses modules: dtCore, dtABC, dtUtil

Purpose
-------
To demonstrate the physics package implemented in Delta3D.

Instructions
------------
Win32 Binary:
- Go to bin.
- Double-click testPhysics.exe.

Linux Binary:
- cd bin
- ./testPhysics

Controls
--------
B  -  Drop crate (box)
C  -  Drop radioactive drum (cylinder)
S  -  Drop a happy sphere (sphere)
P  -  Pause
Esc - Exit

Left   Mouse Button Drag - Rotate camera
Right  Mouse Button Drag - Translate camera
Middle Mouse Button Scroll - Zoom camera
Left   Mouse Button Click - "Pick" an object
