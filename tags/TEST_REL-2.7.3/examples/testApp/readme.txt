Uses modules: dtCore, dtABC, dtUtil

Purpose
-------
To demonstrate basic functionality of the dtABC::Application class,
i.e. creating a simple application and setting basic camera parameters.

Instructions
------------
Win32 Binary:
- Go to bin.
- Double-click testApp.exe.

Linux Binary:
- cd bin
- ./testApp

Controls
---------
Esc - exit
