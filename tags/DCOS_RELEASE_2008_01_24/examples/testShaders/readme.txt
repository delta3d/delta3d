# Delta3D Examples: testShaders        #
#                                      #
########################################

Uses modules: dtCore, dtABC, dtUtil

Purpose
-------
To provide an example of how to add a shader
to a dtCore::Object.

Instructions
------------
Win32 Source: 
- Open VisualStudio\examples\testShaders\testShaders.sln.
- Build testShaders.
- Start testShaders.

Win32 Binary:
- Go to bin.
- Double-click testBumpMap.exe.

Linux Source:
- To build only testShaders, from the root delta3d folder:
  scons bin/testShaders
  
Linux Binary:
- cd bin
- ./testShaders

Controls
--------
Space - Toggle Shader On/Off
Esc   - Exit
