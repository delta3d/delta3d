Uses modules: dtCore, dtABC

Purpose
-------
To demonstrate basic functionality of the dtABC::Application class,
i.e. creating a simple application and setting basic camera parameters.

Instructions
------------
Win32 Source: 
- Open examples/examples.sln.
- Build testApp.
- Start testApp.

Win32 Binary:
- Go to examples/testApp/Release.
- Double-click testApp.exe.

Linux Source:
- Run 'cmake .' from examples/ directory.
- To build all examples: run 'make' from examples/
- To build only testApp: run 'make' from examples/testApp.
- Enter examples/testApp directory.
- Run 'testApp'.

Controls:

Esc - exit
