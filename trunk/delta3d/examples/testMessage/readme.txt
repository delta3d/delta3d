#                                      #
########################################

Uses modules: dtCore

Purpose
-------
To demonstrate the message passing functionality of the Base class using
Sarah Thompson's sigslot.h implementation of the signals/slots algorithm.

Instructions
------------
Win32 Source: 
- Open examples/examples.sln.
- Build testMessage.
- Start testMessage.

Win32 Binary:
- Go to examples/testMessage/Release.
- Double-click testMessage.exe.

Linux Source:
- Run 'cmake .' from examples/ directory.
- To build all examples: run 'make' from examples/
- To build only testMessage: run 'make' from examples/testMessage.
- Enter examples/testMessage directory.
- Run 'testMessage'.

Controls
--------
Esc  -  Exit
