# Delta3D Examples: testInput          #
#                                      #
########################################

Uses modules: dtCore, dtABC, dtUtil

Purpose
-------
To demonstrate basic functionality of the LogicalInputDevice class,
specifically its ability to map user input to various buttons and
axes.

Instructions
------------
Win32 Source: 
- Open VisualStudio\examples\testInput\testInput.sln.
- Build testInput.
- Start testInput.

Win32 Binary:
- Go to bin.
- Double-click testInput.exe.

Linux Source:
- To build only testInput, from the root delta3d folder:
  scons bin/testInput

Linux Binary:
- cd bin
- ./testInput

Controls
--------
Esc  -  Exit
