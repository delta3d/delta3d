Setup


Figure 1.1: STAGE Startup

      The STAGE application should look something like Figure 1.1 on startup, please be aware that some windows and tools may be placed differently for you than in the figure. Most windows can be moved and organized, feel free to experiment. By default, STAGE should load the PortSC map found in the data/maps directory. If this does not happen, then you will have to manually set the context directory. The context directory is the main directory that STAGE will search for all resources such as maps and meshes.


Figure 1.2: Changing our Context Directory.

Figure 1.3: Resources Found.

      To set your Context directory, go to “File->Change Project…” as shown in Figure 1.2. Clicking this will display a new pop up window that will allow you to specify the directory you wish. The Context directory we want is the “data” folder located within the install directory (“C:\Port Security\data” if you used the default install location). Once a Context directory has been properly set, the Prefabs browser window should be populated with resources it has found as shown in Figure 1.3.
