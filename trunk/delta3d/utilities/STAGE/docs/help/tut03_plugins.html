Plugins

      Plugins are additional tools that can be used to modify the functionality of STAGE such as what is shown in a window, the keyboard and mouse controls to manipulate objects and the camera, and can even be used to add completely new features and functionality. To manage your available Plugins, select “Edit->Manage Plugins…” from the menu bar. This option will bring up a new window that will allow you to enable or disable the available Plugins (figure 3.1). There are a few Plugins provided with STAGE, however, new Plugins can be written and added on as desired by the user. Information about the Plugins provided by STAGE will be discussed later in this tutorial in the "Custom Tools" section.
