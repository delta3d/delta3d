Maps

      We are now ready to begin map creation! There are two ways of doing this. The first is to begin with a completely new and empty map, and the second is to load and edit an existing map.

      To create a new empty map, click on “File->New Map…” in the main menu bar. This will prompt you to specify a name and description for your new map. After you have done this, your new map will be visible in the main editor views. It should look like a plain blue screen with a floating cube somewhere ( figure 2.1 below).
 
 


Figure 2.1: An Empty Map.

      To load an existing map, click on “File->Open Map…” from the menu bar. A window will pop up allowing you to specify an available map to open. Remember, maps can only be found if you have a proper context directory set and there are maps in the context directory "maps" folder. Choose a map from the list and press OK to load that map.
