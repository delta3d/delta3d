Uses modules: dtCore, dtABC

Purpose
-------
To demonstrate basic functionality of the UIDrawable GUI class. A SliderBar
controls the opacity of the box. "Delta3D" is displayed inside a TextBox.
An EditableTextBox can be... edited. The Button exits the application. The
cursor also changes to a different icon within the window.

Instructions
------------
Win32 Source: 
- Open examples/examples.sln.
- Build testGUI.
- Start testGUI.

Win32 Binary:
- Go to examples/testGUI/Release.
- Double-click testGUI.exe.

Linux Source:
- Run 'cmake .' from examples/ directory.
- To build all examples: run 'make' from examples/
- To build only testGUI: run 'make' from examples/testGUI.
- Enter examples/testGUI directory.
- Run 'testGUI'.

Controls
--------
Esc  -  Exit
