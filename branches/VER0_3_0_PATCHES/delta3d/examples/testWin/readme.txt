Uses modules: dtCore

Purpose
-------
To demonstrate the gui_fl, which allows a user to inspect and change some 
aspects of a Delta3D window.

Instructions
------------
Win32 Source: 
- Open examples/examples.sln.
- Build testWin.
- Start testWin.

Win32 Binary:
- Go to examples/testWin/Release.
- Double-click testWin.exe.

Linux Source:
- Run 'cmake .' from examples/ directory.
- To build all examples: run 'make' from examples/
- To build only testWin: run 'make' from examples/testWin.
- Enter examples/testWin directory.
- Run 'testWin'.

Controls
--------
Mouse moves cessna (with box child) on a horizontal plane.
