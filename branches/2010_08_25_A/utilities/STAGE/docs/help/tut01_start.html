Getting Started


Figure 1.1: STAGE Startup.

      The STAGE application should look something like Figure 1.1 on startup, please be aware that some windows and tools may be placed differently for you than in the figure.
 
 


Figure 1.2: Changing your Context Directory.

Figure 1.3: Resources Found.

      The first step is to set up your context. The context is the directory in which STAGE will find all resources such as maps and meshes. To set your Context directory, go to “File->Change Project…” as shown in Figure 1.2. Clicking this will display a new pop up window that will allow you to specify the directory you wish. A Context directory should contain all of the resources you wish to use, each within their own sub-directory (e.g. Maps are in the "Maps" folder, Static Meshes are in the "StaticMeshes" folder, etc). Once a Context directory has been properly set, all resource browsers should be populated with resources it has found (Figure 1.3 shows the prefab browser with resources loaded).
