# Delta3D Examples: testCharacters     #
#                                      #
########################################

Uses modules: dtCore, dtChar, dtABC

Purpose
-------
To demonstrate basic functionality of the ReplicantBody/Cal3D character
animation toolkit.

Instructions
------------
Win32 Source: 
- Open examples/examples.sln.
- Build testCharacters.
- Start testCharacters.

Win32 Binary:
- Go to examples/testCharacters/Release.
- Double-click testCharacters.exe.

Linux Source:
- Run 'cmake .' from examples/ directory.
- To build all examples: run 'make' from examples/
- To build only testCharacters: run 'make' from examples/testCharacters.
- Enter examples/testCharacters directory.
- Run 'testCharacters'.

Controls
--------
Up     -  Walk marine forward
Left   -  Turn marine left
Right  -  Turn marine right
Shift  -  Hold down to run
Esc    -  Exit

Left Mouse Button    -  Rotate View
Right Mouse Button   -  Translate View
Middle Mouse Button  -  Zoom View
