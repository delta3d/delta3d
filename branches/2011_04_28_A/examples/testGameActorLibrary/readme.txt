Uses modules: dtActors, dtCore, dtDAL, dtGame

Purpose
-------

The testGameActorLibrary project will generate a library containing some 
example actors.  The actors, while not terribly useful, can be loaded into
STAGE or any other GameManager-based application.

The testGameActorLibrary is not used by itself and mostly lives to
support the UnitTests and as a coding example.
