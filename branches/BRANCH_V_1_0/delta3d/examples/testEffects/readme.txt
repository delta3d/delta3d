########################################

Uses modules: dtCore, dtABC

Purpose
-------
To demonstrate particle effects. This example shows a smoke particle effect as
well as an explosion particle effect.

Instructions
------------
Win32 Source: 
- Open \VisualStudio\examples\testEffects\testEffects.sln.
- Build testEffects.
- Start testEffects.

Win32 Binary:
- Go to examples/testEffects/Release.
- Double-click testEffects.exe.

Linux Source:
- Run 'cmake .' from examples/ directory.
- To build all examples: run 'make' from examples/
- To build only testEffects: run 'make' from examples/testEffects.
- Enter examples/testEffects directory.
- Run 'testEffects'.

Controls
--------
Up     -  Rotate camera up
Down   -  Rotate camera down
Left   -  Rotate camera left
Right  -  Rotate camera right
Esc    -  Exit
